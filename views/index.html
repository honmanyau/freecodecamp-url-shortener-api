URL Shortener

This is a simple URL shortening microservice that takes a valid URL and returns a (hopefully!) shorter URL that redirects to the original URL supplied. The URL shortener is also available as an API (example below). The API only accepts URLs with either the HTTP or HTTPS protocol identifiers; he server attempts to resolve a URL using the HTTPS protocol if it is not specified.

It is worth noting that the parameter returned in a URL shortened by this microservice is base-36 encoded to make them shorter; base-36 encoding was chosen as it is the largest radix supported by the toString() and parseInt() methods in vanilla Javascript.

Example API Call:

https://kurz.glitch.me/api?url=https://lithe.co

Example Output:

{"target":"https://lithe.co","shortened":"https://kurz.glitch.me/0"}
